Source: "ידיעונים, שנתונים ולוח שנה אקדמית / המכללה האקדמית להנדסה בראודה בכרמיאל", שנתון תשפ"ד 2023-2024.
Department chapter: "המחלקה להנדסת מערכות מידע – תשפ"ד, רפורמה 2020" — printed pages 147-158 (PDF page-index 146-157, 0-based; printed page = index + 1).

Method: every page below was rendered from the PDF at 3x zoom and visually inspected for underline formatting in the "קורסי קדם וקורסים צמודים" (prerequisite/corequisite) column. Per the document's own legend (p.149): **"קו תחתי - קורס צמוד או מקביל"** ("underline = attached/corequisite or parallel course"). A plain (non-underlined) course code/name in that column is a strict prerequisite; an underlined one is a corequisite.

Legend for table columns: ה = lecture hours, ת = tutorial hours, מ = lab hours, פ = project hours, נ"ז = credits.
