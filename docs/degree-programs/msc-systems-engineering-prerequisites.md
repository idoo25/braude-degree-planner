## Key finding: no prerequisite/corequisite column in this program's tables

Both course tables in this chapter (recommended program, printed p. 210; elective courses,
printed p. 211) were rendered at 3x resolution and visually inspected.

- **Mandatory/recommended-sequence table** (p. 210) columns: שנה וסמסטר (year+semester),
  שם הקורס (course name), מספר קורס (course number), שעות הרצאה (lecture hours), שעות
  שבועיות (weekly hours), נקודות זכות (credits), and a per-block סך הכול (subtotal).
- **Elective table** (p. 211) columns: התחום (track: ניהול/הנדסה), שם הקורס, מספר קורס,
  שעות הרצאה, שעות שבועיות, נקודות זכות.

**Neither table has a "קורסי קדם וקורסים צמודים" column at all**, and no course code or name
anywhere in this chapter is underlined (the document's convention for marking a corequisite /
"קורס צמוד"). A full-text scan of the chapter for "קדם" and "צמוד" found zero real hits inside
the tables (the only "קדם" substring match in the surrounding chapter text is inside the
unrelated word "כדרישת קדם" discussed below, and inside prose words like "מתקדם"/"להתקדם").

Consequently, **no strict-prerequisite vs. corequisite distinction could be extracted for this
program's course tables**, because the source document does not encode one here. This is
reported as the actual (empty) finding rather than inferred or fabricated.

There is, however, one real program-level prerequisite requirement (not a table cell): specific
undergraduate-level "leveling" courses are a precondition for participating in the program's
mandatory courses if missing from the applicant's transcript (see Footnotes below).

## Summary

- Mandatory/recommended-sequence courses: **12** (across 4 semesters), totaling 41 נ"ז.
- Elective courses listed: **9** (4 Management-track, 5 Engineering-track); students pick 1 from
  each track (2 total).
- Corequisite ("קורס צמוד", underlined) pairs found: **0** — no prerequisite/corequisite column
  exists in either table in this program.
- One real prerequisite mechanism found: 4 named undergraduate-level leveling courses are a
  program-entry (not per-course) prerequisite for students missing them from their transcript.
