## Chapter structure (this is NOT an 8-semester engineering program)

This chapter is organized in three distinct units, run by "היחידה להוראה" (the Teaching Unit),
which is also responsible for General Studies and the Sport unit:

- **א. לימודי הוראה (Teaching Studies)** — printed pp. 180–189. Teacher-certification tracks
  (תעודת הוראה), not a standard degree-sequence. Contains:
  - Program description, admission conditions, study requirements, recognition of prior
    learning, and the internship/stאז' track (pp. 180–182, prose only, no course tables).
  - **Five course tables with a "קורסי קדם" (prerequisite courses) column** — the actual
    prerequisite/corequisite data extracted below (pp. 183–187).
  - One further course table, "קורסי בחירה לתעודת הוראה במקצוע הנדסי" (elective courses for
    the engineering-subject certificate, pp. 187–188) — this table has **no** קורסי קדם column
    at all (its columns are only ש"ש / נ"ז / שם הקורס / מס' הקורס / אשכולות-cluster).
- **ב. לימודים כלליים (General Studies)** — printed pp. 189–192. A flat list of ~60 general-
  education elective courses (מס' הקורס / שם הקורס / נ"ז only) that all B.Sc. engineering
  students must take 6 נ"ז of. No prerequisite column.
- **ג. לימודי חינוך גופני ספורט (Physical Education / Sport)** — printed pp. 192–196. Unit
  description, mandatory PE credit rules, varsity-team ("נבחרות ייצוגיות") and scholarship
  rules, and a flat list of ~35 sport course offerings (מס' קורס / מיקום / נ"ז / ש"ש / תאור).
  No prerequisite column.

**Only unit א (לימודי הוראה) contains prerequisite/corequisite data**, in 5 of its course
tables. Units ב and ג, and the 6th table in unit א (electives), list courses with no
prerequisite/corequisite column whatsoever — they are reported below for completeness of the
chapter description, without a fabricated prerequisites/corequisites split.

## Key finding: no underline formatting is used anywhere in this chapter — corequisites are marked by an explicit text note instead

All 5 course tables containing a "קורסי קדם" column were rendered as page images at 3x and 6x
zoom (printed pages 183, 184, 185, 186, 187 — 0-indexed 182–186) and visually inspected cell by
cell. **No course code or course name anywhere in this chapter is underlined.** This differs
from the engineering-program convention described for this document (underline = corequisite).

Instead, this chapter marks a corequisite-like relationship (a course that "may be studied
concurrently" with a listed prerequisite, matching the official definition of "קורס צמוד" in
the academic-procedures chapter: "קורס שאפשר להשתתף בו רק אם הקורס שאליו הוא מוצמד נלקח באותו
סמסטר") by appending a **literal parenthetical note directly under the course reference** in
the same table cell:
- "(ניתן ללמוד במקביל)" — "(may be studied concurrently)", or
- "(ניתן ללמוד גם במקביל)" — "(may also be studied concurrently)".

Also note: the column header in every table here is simply **"קורסי קדם"** (prerequisite
courses) — never "קורסי קדם וקורסים צמודים" as in the engineering-program template.

In the tables below, any prerequisite course carrying this parenthetical note is listed under
**Corequisites** (bold source note reproduced verbatim); everything else is listed under
**Strict prerequisites**.

## General definition of "קורס צמוד" (corequisite), for reference

This definition is not local to this chapter — it is found in the academic-procedures chapter
of the same yearbook (.txt line ~15225) and is quoted here as the governing rule for what a
"corequisite" means throughout the document: **"קורס צמוד - קורס שאפשר להשתתף בו רק אם הקורס
שאליו הוא מוצמד נלקח באותו סמסטר"** ("a corequisite is a course you may only attend if the
course it's attached to is taken in the same semester"). In this chapter's own tables, that
relationship is expressed via the parenthetical notes "(ניתן ללמוד במקביל)" /
"(ניתן ללמוד גם במקביל)" rather than underlining — see "Key finding" above.

## Summary

- 5 course tables with a קורסי קדם column found (Tables 1–5 above), covering the required
  courses for: the shared teaching-certificate core (Table 1), the engineering-subject
  certificate (Table 2), the math certificate (Table 3), the combined engineering+math
  certificate (Table 4), and the math-certification extension (Table 5).
- Combined unique course rows across Tables 1–5: **~40** (many courses, e.g. 81399, 81375,
  81553, 81558, 81410, 81411, recur across multiple tables/tracks).
- Corequisite relationships found (marked by the "(ניתן ללמוד \[גם\] במקביל)" note, not
  underline): **7** — 81375/81285 → 81399 (Table 1); 81553 → 81399+81375 (Tables 3, 4, 5);
  81558 → 81399+81375 (Tables 3, 4, 5). Counting each distinct course-pair once per table as
  listed above.
- No underline formatting was found anywhere in this chapter's tables.
- One further course table (Table 6, electives) and two further whole units (General Studies;
  Sport) exist in this chapter but carry **no** prerequisite/corequisite column at all — reported
  as such rather than forced into a prerequisites/corequisites split.
