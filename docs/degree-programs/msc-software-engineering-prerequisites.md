212–215 (raw 0-indexed PDF pages 211–214).

Chapter: לימודים לתואר מוסמך במדעים M.Sc. → תואר שני .M.Sc בהנדסת תוכנה (TOC page 212). This
is the separate **Master's** chapter, distinct from the B.Sc. Software Engineering department

## Key finding: no prerequisite/corequisite column in this program's tables

Both course tables in this chapter (recommended program, printed p. 214; elective courses,
printed p. 215) were rendered at 3x resolution and visually inspected.

- **Mandatory/recommended-sequence table** (p. 214, four sub-tables by semester) columns:
  שם הקורס (course name), מספר קורס (course number), שעות הרצאה (lecture hours), שעות
  שבועיות (weekly hours), נקודות זכות (credits).
- **Elective table** (p. 215) columns: שם הקורס, מספר קורס, שעות הרצאה, שעות שבועיות,
  נקודות זכות.

**Neither table has a "קורסי קדם וקורסים צמודים" column at all**, and no course code or name
anywhere in this chapter is underlined (the document's convention for marking a corequisite /
"קורס צמוד"). A full-text scan of the chapter for "קדם" and "צמוד" found no real hits inside the
tables (the only "קדם" substring matches nearby are inside unrelated prose words like
"להתקדם"/"מתקדם", not a column header).

Consequently, **no strict-prerequisite vs. corequisite distinction could be extracted for this
program's course tables**, because the source document does not encode one here.

## Admission / program description (verbatim excerpts)

**מטרת התוכנית (Program goal):** "להעניק לבעלי תואר ראשון... ידע מדעי מחקרי ועדכני לפתרון
בעיות הנדסיות מציאותיות ודרכים ליישום הידע בעבודת המהנדס... מיומנויות בהובלה ובניהול של
פרויקטי תוכנה גדולים ובביצוע מחקרים מדעיים בהנדסת תוכנה."

**קורסי החובה (Mandatory courses), per program description text:** "מתודולוגיות מתקדמות
לפתוח תוכנה, מערכות לומדות, הבטחת איכות תוכנה ושיפור תהליכים, ניהול פרויקטי תוכנה, ניהול
צוותי פיתוח תוכנה ועבודת גמר."

**Program structure / leveling courses (general mention, no course-code list found):**
"תוכנית הלימודים המומלצת נמשכת שנתיים... סטודנטים יוכלו לפרוס את הלימודים גם לאורך שלוש
שנים, כדי להקל על מי שעובד במשרה מלאה או על מי שחייב בקורסי השלמה." Unlike the Systems
Engineering M.Sc. chapter, no specific named leveling courses or credit values are listed here —
only this general reference to "קורסי השלמה" for students who need them.

**מספר קורסי הבחירה (Number of electives offered):** "עשוי להשתנות משנה לשנה וייקבע לפי
כמות הסטודנטים הלומדים באותה שנה." (May vary year to year based on enrollment.)
